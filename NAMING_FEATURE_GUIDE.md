---

## 📋 功能概述

本功能允许用户在首次使用小程序时，为自己的 AI 陪伴助手取一个专属名字。名字会保存在微信本地存储中，无需服务器支持，下次启动时自动使用用户设置的名字。

### 核心特性

- ✅ **首次启动引导**：未设置名字时自动跳转到命名页面
- ✅ **温暖治愈 UI**：粉色调、圆角设计、可爱动画
- ✅ **推荐名字**：提供 8 个温暖系名字供用户快速选择
- ✅ **本地存储**：使用 `wx.setStorageSync` 保存，无需服务器
- ✅ **全局生效**：所有页面的 AI 称呼自动同步更新
- ✅ **随时修改**：用户可在个人中心重置名字

## 🔧 技术实现

### 1. 命名引导页面 (`pages/naming/`)

#### 页面功能
- 欢迎语和引导文案
- 输入框（2-4 个字限制）
- 8 个推荐名字按钮（小树/念念/陪陪/暖暖/安安/心心/瓜瓜/泡泡）
- 确认按钮（输入有效时激活）

#### UI 设计
- **主题色**：粉色系（#FF6B81, #FF8FA3, #FFE4E9）
- **风格**：温暖治愈、圆角卡片、可爱动画
- **交互**：点击推荐名字自动填充，确认时显示成功弹窗

### 2. 全局名字管理工具 (`utils/ai-name.js`)

提供 4 个核心方法：

```javascript
// 获取 AI 名字（未设置时返回默认值"瓜瓜"）
aiNameUtil.getName()

// 保存 AI 名字（返回是否成功）
aiNameUtil.setName('小树')

// 重置为默认名字
aiNameUtil.resetName()

// 检查是否已设置名字
aiNameUtil.hasName()
```

### 3. 首次启动检测 (`app.js`)

```javascript
onLaunch() {
  this.checkAiName();  // 检查名字设置
  // ...
}

checkAiName() {
  if (!aiNameUtil.hasName()) {
    // 未设置名字，跳转到命名页面
    wx.reLaunch({ url: '/pages/naming/index' });
  } else {
    // 已设置，读取名字到全局数据
    this.globalData.aiName = aiNameUtil.getName();
  }
}
```

### 4. 页面动态名字替换

所有页面统一模式：

```javascript
// 1. 引入工具
const aiNameUtil = require('../../utils/ai-name.js');

// 2. 页面数据中添加 aiName
data: {
  aiName: '瓜瓜'
}

// 3. 加载时读取
loadAiName() {
  const name = aiNameUtil.getName();
  this.setData({ aiName: name });
}
```

```xml
<!-- WXML 中使用 -->
<text>{{aiName}}在这里听你说</text>
```

## 📝 使用流程

### 首次使用
1. 用户打开小程序
2. 自动检测未设置名字
3. 跳转到命名引导页面
4. 用户输入名字或选择推荐
5. 点击确认，显示成功弹窗
6. 自动跳转到首页，开始使用

### 再次启动
1. 用户打开小程序
2. 检测到已有名字
3. 直接进入首页
4. 所有页面显示用户设置的名字

## 🐛 注意事项

1. **存储限制**：微信本地存储上限 10MB，名字仅占极少空间
2. **兼容性**：支持微信小程序基础库 2.0+
3. **性能影响**：名字读取为同步操作，对性能无影响
4. **隐私安全**：名字仅保存在用户本地，不会上传服务器
