> 温暖治愈系陪伴助手 - 为女王大人定制

---

## 📱 项目简介

瓜瓜陪伴是一款温暖治愈系的微信小程序，旨在为用户提供情感陪伴和心灵慰藉。

### 核心功能

1. **🏠 首页** - 温暖的主菜单，快速访问所有功能
2. **💬 树洞倾诉** - 与瓜瓜聊天，倾诉心事
3. **✨ 今日亮点** - 记录每日成就和美好瞬间
4. **📖 睡前故事** - 温暖的故事伴你入眠
5. **🧘 放松引导** - 呼吸引导和放松练习
6. **👤 个人中心** - 查看统计数据和设置

---

## 🛠️ 技术栈

- **微信小程序** - 原生开发（WXML/WXSS/JavaScript）
- **Vant Weapp** - 组件库（待集成）
- **本地存储** - wx.Storage 数据持久化

## 🚀 快速开始

### 环境要求

- 微信开发者工具（最新版本）
- 微信小程序账号（用于真机测试）

### 安装步骤

1. **克隆项目**
   ```bash
   cd /root/.openclaw/workspace/guagua-miniprogram
   ```

2. **导入项目**
   - 打开微信开发者工具
   - 选择"导入项目"
   - 选择项目目录：`/root/.openclaw/workspace/guagua-miniprogram`
   - 填写 AppID（测试可选"测试号"）

3. **安装依赖**
   ```bash
   # 安装 Vant Weapp 组件库
   npm install @vant/weapp -S --production
   ```

4. **构建 npm**
   - 微信开发者工具 → 工具 → 构建 npm

5. **运行项目**
   - 点击编译按钮即可在开发者工具中预览

## 📝 功能说明

### 1. 树洞倾诉

- **功能**: 聊天界面，用户可与瓜瓜倾诉心事
- **实现**: 本地关键词匹配回复（不使用付费 API）
- **数据存储**: 聊天记录保存在本地存储

### 2. 今日亮点

- **功能**: 记录每日成就和心情
- **实现**: 成就列表、心情统计、鼓励语
- **数据存储**: 成就记录保存在本地存储

### 3. 睡前故事

- **功能**: 故事列表和播放
- **状态**: Placeholder 数据（待千夜交付故事内容）
- **待对接**: 音频播放功能

### 4. 放松引导

- **功能**: 呼吸引导动画和倒计时
- **实现**: 4-7-8 呼吸法、5 分钟倒计时
- **模式**: 呼吸练习、身体放松、冥想引导

### 5. 个人中心

- **功能**: 用户信息、统计数据、设置
- **实现**: 陪伴天数、聊天次数、成就数量统计
