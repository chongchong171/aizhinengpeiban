### 用户体验
1. **首次启动自动引导**：未设置名字时强制跳转到命名页面
2. **推荐名字快速选择**：8 个温暖系名字（小树/念念/陪陪/暖暖/安安/心心/瓜瓜/泡泡）

5. **全局同步**：所有页面自动显示用户设置的名字

## 🔧 使用方法

### 测试流程
1. 清除小程序本地存储（或首次安装）
2. 打开小程序
3. 自动跳转到命名页面
4. 输入名字或点击推荐
5. 点击确认
6. 查看首页、聊天页等是否显示新名字

### 代码示例
```javascript
// 在任何页面中获取 AI 名字
const aiNameUtil = require('../../utils/ai-name.js');
const name = aiNameUtil.getName();  // 返回用户设置的名字或"瓜瓜"

// 保存新名字
aiNameUtil.setName('小树');  // 返回 true/false

// 检查是否已设置
if (aiNameUtil.hasName()) {
  console.log('已有名字');
}
```
