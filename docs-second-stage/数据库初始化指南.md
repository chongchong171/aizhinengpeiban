# 数据库初始化指南

**日期：** 2026-03-16  
**适用项目：** 瓜瓜陪伴微信小程序

---

## 📊 需要创建的集合

在微信云开发控制台，创建以下两个集合：

### 1. dormitories（宿舍集合）

**权限设置：**
- 所有用户：可读
- 仅创建者：可写自己的记录

**字段说明：**

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| name | String | ✅ | 宿舍名称 |
| style | String | ✅ | 风格（温馨/可爱/简约） |
| creator | String | ✅ | 创建者 openid |
| createdAt | Number | ✅ | 创建时间戳 |
| memberCount | Number | ✅ | 当前人数 |
| capacity | Number | ✅ | 容量（4/6/8） |
| ageGroup | String | ✅ | 年龄段 |
| introduction | String | ❌ | 宿舍介绍 |
| agreement | String | ❌ | 宿舍公约 |
| activeDays | Number | ✅ | 活跃天数 |
| level | Number | ✅ | 等级 |
| exp | Number | ✅ | 经验值 |
| lastActiveDate | Number | ✅ | 最后活跃时间 |

**索引建议：**
- creator（升序）
- createdAt（降序）

---

### 2. members（舍友集合）

**权限设置：**
- 所有用户：可读
- 仅本人：可写自己的记录

**字段说明：**

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| dormitoryId | String | ✅ | 宿舍 ID |
| userId | String | ✅ | 用户 openid |
| nickname | String | ✅ | 昵称 |
| title | String | ✅ | 自动分配称呼 |
| customTitle | String | ❌ | 自定义称呼 |
| joinedAt | Number | ✅ | 加入时间戳 |
| activeDays | Number | ✅ | 活跃天数 |
| moodCount | Number | ✅ | 分享心情次数 |
| goodnightCount | Number | ✅ | 晚安打卡次数 |
| lastActiveDate | Number | ✅ | 最后活跃时间 |

**索引建议：**
- userId（升序）
- dormitoryId（升序）
- dormitoryId + joinedAt（复合索引，升序）

---

## 🔧 创建步骤

### 方法一：微信开发者工具

1. 打开微信开发者工具
2. 点击"云开发"按钮
3. 进入"数据库"
4. 点击"+"创建集合
5. 输入集合名称：`dormitories`
6. 设置权限（自定义规则）
7. 重复步骤创建 `members` 集合

### 方法二：云开发控制台

1. 访问 https://cloud.weixin.qq.com
2. 选择对应的小程序
3. 进入"数据库"
4. 点击"添加集合"
5. 输入集合名称
6. 设置权限

---

## 🔐 权限规则示例

### dormitories 集合权限

```json
{
  "read": "auth.openid != null",
  "write": "auth.openid == resource.data.creator"
}
```

### members 集合权限

```json
{
  "read": "auth.openid != null",
  "write": "auth.openid == resource.data.userId"
}
```

## 🐛 常见问题

### Q: 创建宿舍失败，提示"集合不存在"
A: 请先在云开发控制台创建集合，然后再测试。

### Q: 权限不足，无法写入数据
A: 检查集合的权限规则是否正确设置。

### Q: 查询速度慢
A: 确保已为常用查询字段创建索引。
