## 🚀 步骤 1：上传云函数（6 个）

1. **创建宿舍**
   ```

   → 上传并部署：云端安装依赖

   ```

2. **获取宿舍首页**
   ```
   右键 cloudfunctions/getDormitoryHome
   → 上传并部署：云端安装依赖
   → 等待完成提示
   ```

3. **加入宿舍**
   ```
   右键 cloudfunctions/joinDormitory
   → 上传并部署：云端安装依赖
   → 等待完成提示
   ```

4. **更新活跃天数**
   ```
   右键 cloudfunctions/updateActiveDays
   → 上传并部署：云端安装依赖
   → 等待完成提示
   ```

5. **更新公约**
   ```
   右键 cloudfunctions/updateAgreement
   → 上传并部署：云端安装依赖
   → 等待完成提示
   ```

6. **更新称呼**
   ```
   右键 cloudfunctions/updateMemberTitle
   → 上传并部署：云端安装依赖
   → 等待完成提示
   ```

### 方法 B：命令行上传（高级）

```bash
# 安装微信云开发 CLI
npm install -g wx-server-sdk

# 进入云函数目录
cd /root/.openclaw/workspace/guagua-github/cloudfunctions

# 逐个上传
for func in createDormitory getDormitoryHome joinDormitory updateActiveDays updateAgreement updateMemberTitle; do
  echo "上传 $func..."
  cd $func
  npm install
  # 需要微信开发者工具配合
  cd ..
done
```

## 📊 步骤 2：创建数据库集合

### 在微信开发者工具中：

1. **打开云开发控制台**
   ```
   顶部菜单 → 云开发 → 点击进入
   ```

2. **创建 dormitories 集合**
   ```
   点击：数据库
   点击：+ 创建集合
   输入：dormitories
   点击：确定
   ```

3. **创建 members 集合**
   ```
   点击：+ 创建集合
   输入：members
   点击：确定
   ```

### 数据库权限设置：

**dormitories 集合：**
```json
{
  "read": "auth.openid == 'OPENID' || true",
  "write": "auth.openid == 'OPENID'"
}
```

**members 集合：**
```json
{
  "read": "auth.openid == 'OPENID' || true",
  "write": "auth.openid == 'OPENID'"
}
```

## ⚠️ 常见问题

### Q1: 上传云函数失败

**错误信息：** `云环境 ID 未配置`

**解决方案：**
```
1. project.config.json 中配置 cloudfunctionRoot
2. 确认已开通云开发
3. 确认云环境 ID 正确
```

### Q2: 创建宿舍失败

**错误信息：** `集合不存在`

**解决方案：**
```
1. 检查是否创建了 dormitories 集合
2. 检查是否创建了 members 集合
3. 检查数据库权限设置
```

### Q3: 看不到新功能

**问题：** 页面还是旧版本

**解决方案：**
```
1. 微信开发者工具 → 清除缓存
2. 勾选"清除全部缓存"
3. 重新编译
4. 如果还不行，删除项目重新导入
```

### Q4: 云函数超时

**错误信息：** `Timeout`

**解决方案：**
```
1. 检查云函数代码是否有死循环
2. 增加云函数超时时间（默认 3 秒）
3. 检查数据库连接是否正常
```
