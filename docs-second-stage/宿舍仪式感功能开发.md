## 📋 功能概述

为瓜瓜陪伴微信小程序开发女生宿舍仪式感功能，增强舍友间的归属感和互动体验。

---

## ✨ 功能清单

### 1. 舍友称呼系统

**功能描述：**
- 宿舍创建者自动获得"大当家"称号
- 第 2 个成员："二当家"
- 第 3 个成员："三当家"
- 后续成员：按顺序（四当家、五当家...）
- 支持自定义称呼（可修改）

**技术实现：**
- 云函数 `joinDormitory` 自动分配称呼
- 工具函数 `utils/dormitory-title.js` 提供称呼管理
- 前端支持编辑自定义称呼

**数据结构：**
```javascript
{
  title: '大当家',      // 自动分配
  customTitle: ''       // 自定义称呼（可选）
}
```

---

### 2. 宿舍公约

**功能描述：**
- 创建宿舍时可选填公约
- 示例："互相鼓励、不说伤人的话、保守秘密"
- 所有舍友可见
- 仅创建者可修改

**技术实现：**
- 宿舍创建页面添加公约输入框
- 云函数 `updateAgreement` 更新公约
- 宿舍首页展示公约卡片

**数据结构：**
```javascript
{
  agreement: '互相鼓励、不说伤人的话、保守秘密'
}
```

---

### 3. 宿舍纪念日

**功能描述：**
- 自动记录宿舍成立日
- 活跃天数统计
- 特殊纪念日提醒（成立 7 天、30 天等）
- 纪念日弹窗庆祝

**技术实现：**
- 云函数 `getDormitoryHome` 计算纪念日信息
- 工具函数 `getAnniversaryMilestone` 提供里程碑判断
- 宿舍首页展示纪念日卡片

**里程碑节点：**
- 当天：🎉 今天是宿舍成立日！
- 7 天：🎊 宿舍成立 7 天啦！
- 30 天：🌙 宿舍成立满月啦！
- 100 天：💯 宿舍成立 100 天！
- 365 天：🎂 宿舍成立一周年！

## 🗄️ 数据库结构

### 宿舍集合 (dormitories)

```javascript
{
  _id: "dormitory_xxx",
  name: "月光小筑",                    // 宿舍名称
  style: "温馨",                        // 风格
  creator: "user_xxx",                  // 创建者 openid
  createdAt: 1710547200000,             // 创建时间戳
  memberCount: 3,                       // 当前人数
  capacity: 6,                          // 容量
  ageGroup: "18-25 岁（大学/职场）",     // 年龄段
  introduction: "温暖的睡前小天地",      // 介绍
  agreement: "互相鼓励、不说伤人的话...", // 公约
  activeDays: 7,                        // 活跃天数
  level: 3,                             // 等级
  exp: 150,                             // 经验值
  lastActiveDate: 1710633600000         // 最后活跃时间
}
```

### 舍友集合 (members)

```javascript
{
  _id: "member_xxx",
  dormitoryId: "dormitory_xxx",         // 宿舍 ID
  userId: "user_xxx",                   // 用户 openid
  nickname: "小美",                      // 昵称
  title: "大当家",                      // 自动分配称呼
  customTitle: "",                      // 自定义称呼
  joinedAt: 1710547200000,              // 加入时间
  activeDays: 7,                        // 活跃天数
  moodCount: 5,                         // 分享心情次数
  goodnightCount: 3,                    // 晚安打卡次数
  lastActiveDate: 1710633600000         // 最后活跃时间
}
```

## 🎨 UI 设计要点

- 主色调：#FF6B81（粉红）


## 📝 使用说明

### 创建宿舍
1. 进入"创建宿舍"页面
2. 填写宿舍名称、风格、人数、年龄段
3. （可选）填写宿舍介绍
4. （可选）填写宿舍公约
5. 点击"创建宿舍"

### 加入宿舍
1. 通过邀请链接或扫码
2. 自动分配称呼（二当家、三当家...）
3. 可查看宿舍公约

### 修改称呼
1. 在宿舍首页点击"我的称呼"旁的✏️图标
2. 输入自定义称呼
3. 确认保存

### 查看纪念日
1. 进入宿舍首页
2. 查看纪念日卡片
3. 特殊纪念日会弹窗庆祝
