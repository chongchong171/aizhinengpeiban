## 📄 页面详细说明

### 1. 树洞倾诉（主打功能）

**页面路径：** `/pages/chat/chat`

**功能描述：**
- 类似微信的聊天界面
- 用户可以与瓜瓜 AI 对话
- 瓜瓜会倾听用户的心声，给予温暖的回应
- 支持情绪识别和鼓励

**使用场景：**
- 工作压力大时想倾诉
- 遇到开心事想分享
- 感到孤独时想有人陪伴

**核心交互：**
```
用户输入 → 发送消息 → 瓜瓜回复 → 情绪分析 → 温暖鼓励
```

---

### 2. 今日总结（合并功能）

**页面路径：** `/pages/summary/summary`

**功能描述：**
- 合并了原有的"心情日记"和"今日亮点"
- 一站式记录每天的心情和成就
- 瓜瓜会给出专属鼓励

**页面模块：**
1. **今日心情** - 显示当前情绪状态
2. **今日亮点** - 记录今天的美好瞬间（可添加）
3. **心情日记** - 写下今天的感受和想法
4. **瓜瓜的鼓励** - 瓜瓜给你的专属鼓励语

**使用场景：**
- 睡前回顾一天
- 记录成长轨迹
- 情绪管理

---

### 3. 睡前故事

**页面路径：** `/pages/stories/stories`

**功能描述：**
- 温暖治愈的睡前故事集合
- 支持分类筛选
- 可配合 TTS 播放

**故事分类：**
- 治愈系
- 成长系
- 爱情系
- 友情系

**使用场景：**
- 睡前放松
- 缓解焦虑
- 温暖入眠

---

### 4. 放松引导

**页面路径：** `/pages/relax/relax`

**功能描述：**
- 呼吸引导动画
- 文字引导放松
- 倒计时功能

**引导类型：**
- 4-7-8 呼吸法
- 渐进式肌肉放松
- 正念冥想

**使用场景：**
- 工作压力大时
- 焦虑紧张时
- 需要平静时

---

### 5. 瓜瓜读给你听（新增）

**页面路径：** `/pages/read/read`

**功能描述：**
- TTS 朗读功能
- 瓜瓜用温暖的声音为你朗读
- 支持多种内容类型

**内容分类：**
1. **睡前故事** - 温暖的故事
2. **文章** - 精选文章
3. **自定义** - 输入任意文本

**播放控制：**
- 播放/暂停
- 停止
- 语速调节（0.5x - 2.0x）
- 音量调节（0 - 100）

**使用场景：**
- 通勤路上听故事
- 睡前听瓜瓜朗读
- 眼睛累了想听书

**技术实现：**
```javascript
// 调用 TTS API
wx.playVoice({
  filePath: ttsAudioUrl
})
```

---

### 6. 朗读功能（新增）

**页面路径：** `/pages/recite/recite`

**功能描述：**
- 用户自己朗读并录音
- 记录你的声音
- 可回听、删除

**内容类型：**
- 诗歌
- 故事
- 文章
- 自定义内容

**核心功能：**
1. **录音** - 长按录音按钮开始录音
2. **播放** - 回听自己的录音
3. **管理** - 删除不需要的录音

**使用场景：**
- 朗读练习
- 声音日记
- 记录美好时刻

**权限要求：**
- 录音权限（scope.record）

## 🚀 使用指南

### 第一次使用

1. **打开小程序** - 进入首页
2. **浏览功能** - 了解各个功能入口
3. **开始使用** - 点击"树洞倾诉"开始与瓜瓜对话

### 日常使用流程

**早晨：**
- 打开小程序，看到瓜瓜的问候
- 查看今日鼓励语

**工作间隙：**
- 感到压力时，进入"树洞倾诉"
- 或者使用"放松引导"做深呼吸

**晚上：**
- 进入"今日总结"记录一天
- 听"瓜瓜读给你听"入睡
- 或者读一个"睡前故事"

---

## 💡 最佳实践

### 情绪管理

1. **及时倾诉** - 有情绪时立即找瓜瓜倾诉
2. **记录亮点** - 每天记录 3 件开心的事
3. **定期放松** - 每天至少做一次放松引导

### 声音功能

1. **听瓜瓜朗读** - 睡前听 10 分钟，帮助入睡
2. **自己朗读** - 记录声音日记，回顾成长
3. **调节语速** - 找到最舒适的收听速度
