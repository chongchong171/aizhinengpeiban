# 瓜瓜读给你听 - 功能说明

## 功能概述

**用户上传文字，瓜瓜 TTS 朗读**

这是一个文字转语音（TTS）朗读功能，用户可以输入任意文本，选择喜欢的声音和语速，让瓜瓜读给你听。

## 技术实现

### 1. 文本输入框（多行）
- 使用微信小程序 `<textarea>` 组件
- 支持最多 1000 字输入
- 自动高度适应内容

### 2. 声音选择（单选）
- 温柔女声 / 磁性男声
- 使用自定义单选按钮样式
- 数据绑定到 `voice` 字段

### 3. 语速选择（单选）
- 慢 / 中 / 快
- 使用自定义单选按钮样式
- 数据绑定到 `speed` 字段

### 4. 播放控制（播放/暂停）
- 播放按钮：启动 TTS 朗读
- 暂停按钮：暂停当前朗读
- 播放状态指示器（动画效果）

### 5. 保存功能（本地存储）
- 使用 `wx.setStorageSync` 保存到本地
- 保存内容包括：标题、内容、声音、语速、时间
- 朗读列表支持加载和删除

### 6. TTS 调用（占位实现）
- 当前为占位实现，模拟播放效果
- 预留接口对接真实 TTS API
- 支持阿里云、腾讯云等 TTS 服务

## 使用方法

1. 在首页点击"瓜瓜读给你听"入口
2. 在文本框中输入想听的内容
3. 选择喜欢的声音（温柔女声/磁性男声）
4. 选择语速（慢/中/快）
5. 点击"播放"开始朗读
6. 点击"保存当前朗读"可保存到朗读列表
7. 在朗读列表中可快速加载已保存的内容
