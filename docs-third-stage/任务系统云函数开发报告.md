### 1. getDailyTasks - 获取每日任务

**路径：** `cloudfunctions/getDailyTasks/index.js`

**功能实现：**
- ✅ 获取每日任务配置（5 个每日任务）
- ✅ 获取每周挑战任务（3 个每周挑战）
- ✅ 获取宿舍集体任务（3 个宿舍任务）
- ✅ 获取用户当前任务进度
- ✅ 计算任务完成状态
- ✅ 标记已领取奖励的任务
- ✅ 返回任务列表（含进度、完成状态、领取状态）

**返回格式：**
```javascript
{
  success: Boolean,
  data: {
    dailyTasks: [...],        // 每日任务列表
    weeklyChallenges: [...],  // 每周挑战列表
    dormitoryTasks: [...],    // 宿舍集体任务列表
    userPoints: {
      starlight: Number,
      crystal: Number
    }
  }
}
```

**任务配置：**
| 任务类型 | 任务 ID | 星光值 | 次数限制 |
|---------|--------|--------|---------|
| 每日任务 | morning_checkin | +10 | 1 次/天 |
| 每日任务 | night_checkin | +20 | 1 次/天 |
| 每日任务 | share_mood | +15 | 3 次/天 |
| 每日任务 | view_roommate_mood | +5 | 5 次/天 |
| 每日任务 | like_roommate_post | +3 | 10 次/天 |
| 每周挑战 | weekly_full_attendance | +100 | 1 次/周 |
| 每周挑战 | weekly_share_master | +80 | 1 次/周 |
| 每周挑战 | weekly_interaction_star | +150 | 1 次/周 |
| 宿舍任务 | dormitory_goodnight | +50 | 1 次/天 |
| 宿舍任务 | dormitory_mood | +30 | 1 次/天 |
| 宿舍任务 | dormitory_night_talk | +40 | 1 次/周 |

### 2. claimTaskReward - 领取任务奖励

**路径：** `cloudfunctions/claimTaskReward/index.js`

**功能实现：**
- ✅ 验证任务完成状态（服务器端验证）
- ✅ 检查奖励是否已领取（防刷机制）
- ✅ **开启事务**（原子操作）
- ✅ 增加用户积分/经验值
- ✅ 更新任务状态为 claimed
- ✅ 记录积分流水（pointsFlow 集合）
- ✅ 检查成就解锁
- ✅ **提交事务**

**返回格式：**
```javascript
{
  success: Boolean,
  message: String,
  data: {
    taskId: String,
    rewardExp: Number,
    rewardPoints: Number,
    rewardType: String,
    newStarlight: Number,
    newLevel: Number,
    achievements: [...]
  }
}
```

**防刷机制：**
- 每日任务：限制每日领取次数
- 每周挑战：限制每周领取次数
- 服务器端验证任务完成条件
- 事务处理保证数据一致性

### 3. checkIn - 签到功能（新建）

**路径：** `cloudfunctions/checkIn/index.js`

**功能实现：**
- ✅ 签到成功发放星光值（+10 基础，+15 连续 3 天，+20 连续 7 天）
- ✅ 更新连续签到天数
- ✅ 检查连续签到成就
- ✅ 创建签到记录（checkInRecords 集合）
- ✅ 记录积分流水

**返回格式：**
```javascript
{
  success: Boolean,
  message: String,
  data: {
    rewardStarlight: Number,
    consecutiveDays: Number,
    bonusType: String,      // base | consecutive_3 | consecutive_7
    newStarlight: Number,
    newLevel: Number,
    achievements: [...]
  }
}
```

**成就系统：**
| 成就 ID | 成就名称 | 要求 | 奖励 |
|--------|---------|------|------|
| checkin_7 | 初入茅庐 | 连续签到 7 天 | +50 星光值 |
| checkin_30 | 持之以恒 | 连续签到 30 天 | +200 星光值 |
| checkin_100 | 铁杆粉丝 | 连续签到 100 天 | +500 星光值 |
