## 🚀 部署步骤

### 步骤 1：创建数据库集合（5 分钟）

#### 1.1 打开微信云开发控制台
1. 打开微信开发者工具
2. 点击「云开发」按钮
3. 进入数据库管理

#### 1.2 创建 couponRedemptions 集合
1. 点击「+」创建集合
2. 集合名称：`couponRedemptions`
3. 权限设置：选择「自定义规则」
4. 输入以下规则：
   ```json
   {
     "read": "auth.openid == doc.userId",
     "write": "auth.openid == doc.userId"
   }
   ```
5. 点击「确定」

#### 1.3 创建 taskRecords 集合
1. 点击「+」创建集合
2. 集合名称：`taskRecords`
3. 权限设置：同上
4. 点击「确定」

#### 1.4 添加索引

**couponRedemptions 集合：**
1. 进入 couponRedemptions 集合
2. 点击「索引管理」
3. 添加以下索引：
   - 复合索引：`userId` (升序) + `status` (升序)
   - 复合索引：`userId` (升序) + `expireAt` (升序)
   - 单字段索引：`createdAt` (降序)

**taskRecords 集合：**
1. 进入 taskRecords 集合
2. 点击「索引管理」
3. 添加以下索引：
   - 复合索引：`userId` (升序) + `taskType` (升序) + `completedAt` (升序)
   - 复合索引：`userId` (升序) + `claimed` (升序) + `completedAt` (升序)
   - 单字段索引：`completedAt` (降序)

## ⚠️ 常见问题

### Q1: 云函数上传失败
**解决：** 
1. 检查网络连接
2. 确认云环境 ID 正确
3. 重新安装依赖：右键 → 「安装依赖」

### Q2: 数据库权限错误
**解决：**
1. 检查集合权限设置
2. 确认规则中包含 `auth.openid == doc.userId`

### Q3: 页面找不到（404）
**解决：**
1. 检查 app.json 是否添加了页面路由
2. 确认页面文件路径正确
3. 重新编译小程序

### Q4: 星光值不增加
**解决：**
1. 检查 members 集合是否有 starlightPoints 字段
2. 检查云函数日志（云开发控制台 → 云函数 → 日志）
3. 确认 taskRecords 集合已创建
