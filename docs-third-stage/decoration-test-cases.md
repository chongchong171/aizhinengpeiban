## 📋 使用说明

- **测试场景：** 描述测试的业务场景
- **测试步骤：** 详细的操作步骤
- **预期结果：** 期望的系统行为
- **实际结果：** 测试执行时记录
- **测试状态：** Pass / Fail / Blocked
- **优先级：** P0（核心）/ P1（重要）/ P2（一般）
