# 数据库初始化脚本 - 女生宿舍商业化功能

**创建日期：** 2026-03-17  
**用途：** 创建新集合并添加索引

---

## 1. members 集合 - 新增字段

在微信云开发控制台 → 数据库 → members 集合，确保以下字段存在：

```javascript
{
  // 新增积分字段
  starlightPoints: Number,      // 当前星光值（默认 0）
  crystalDiamonds: Number,      // 当前晶钻（默认 0）
  totalStarlight: Number,       // 累计星光值（默认 0）
  totalCrystal: Number,         // 累计晶钻（默认 0）
  
  // 原有字段（确保存在）
  userId: String,               // 用户 openid
  dormitoryId: String,          // 宿舍 ID
  nickname: String,             // 昵称
  avatar: String,               // 头像
  
  // 原有经验值字段
  personalExp: Number,          // 个人经验值
  totalExp: Number,             // 累计经验值
  level: Number,                // 个人等级
  
  // 活跃字段
  consecutiveDays: Number,      // 连续活跃天数
  lastCheckinDate: Number,      // 最后签到日期戳
  
  // 任务进度
  dailyTasks: Object,           // 每日任务进度
  weeklyTasks: Object,          // 每周任务进度
  achievements: Array           // 已解锁成就列表
}
```

**注意：** 已有用户记录会自动获得默认值 0，无需额外操作。

## 2. couponRedemptions 集合 - 新建

在微信云开发控制台 → 数据库 → 创建集合 `couponRedemptions`

**字段结构：**
```javascript
{
  _id: String,                  // 记录 ID（自动生成）
  userId: String,               // 用户 openid
  dormitoryId: String,          // 宿舍 ID
  couponType: String,           // 优惠券类型（如：coupon_50_200）
  couponValue: Number,          // 优惠券面额（元）
  minPurchase: Number,          // 使用门槛（元）
  pointsType: String,           // 积分类型：starlight | crystal
  pointsCost: Number,           // 花费积分数量
  status: String,               // 状态：unused | used | expired
  expireAt: Number,             // 过期时间戳
  usedAt: Number,               // 使用时间戳（可选）
  createdAt: Number             // 创建时间戳
}
```

**索引设置（重要！）：**
1. 创建复合索引：`userId + status`（升序 + 升序）
2. 创建复合索引：`userId + expireAt`（升序 + 升序）
3. 创建单字段索引：`createdAt`（降序）

**权限设置：**
- 所有用户：可读（仅自己的数据）
- 仅管理员：可写

## 3. taskRecords 集合 - 新建

在微信云开发控制台 → 数据库 → 创建集合 `taskRecords`

**字段结构：**
```javascript
{
  _id: String,
  userId: String,
  taskId: String,               // 任务 ID
  taskType: String,             // daily | weekly | achievement
  taskCategory: String,         // daily | weekly | dormitory
  reward: Number,               // 奖励数量
  rewardType: String,           // starlight | crystal | exp
  completedAt: Number,          // 完成时间戳
  claimed: Boolean,             // 是否已领取奖励
  claimedAt: Number             // 领取时间戳
}
```

**索引设置（重要！）：**
1. 创建复合索引：`userId + taskType + completedAt`（升序 + 升序 + 升序）
2. 创建复合索引：`userId + claimed + completedAt`（升序 + 升序 + 升序）
3. 创建单字段索引：`completedAt`（降序）

**权限设置：**
- 所有用户：可读（仅自己的数据）
- 仅管理员：可写

## 4. 数据库权限配置示例

在微信云开发控制台 → 数据库 → 权限管理，设置以下权限：

### members 集合
```json
{
  "read": "auth.openid == doc.userId",
  "write": "auth.openid == doc.userId"
}
```

### couponRedemptions 集合
```json
{
  "read": "auth.openid == doc.userId",
  "write": "auth.openid == doc.userId"
}
```

### taskRecords 集合
```json
{
  "read": "auth.openid == doc.userId",
  "write": "auth.openid == doc.userId"
}
```
