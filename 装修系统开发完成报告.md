## 📊 技术实现

### 布局方案
- **Grid 布局：** 装修网格（8x8）、商品列表（2 列）、排行榜（列表）
- **Flex 布局：** 工具栏、信息卡片、弹窗
- **相对定位：** 装饰品放置、拖拽效果

### 核心交互
```javascript
// 拖拽实现
onDecorationLongPress → startDrag
onTouchMove → updatePosition
onTouchEnd → placeDecoration

// 旋转实现
rotateDecoration → (rotation + 90) % 360

// 购买流程
selectProduct → checkPoints → confirmBuy → updateInventory
```

### 数据模拟
- 装饰品数据（20+ 个商品）
- 排行榜数据（100 条模拟）
- 用户积分（星光值、晶钻）
