## 推荐尺寸

- **图标尺寸：** 512x512px 或 1024x1024px
- **格式：** PNG（透明背景）或 JPG（白色背景）
- **风格：** 2D 卡通、扁平化、简洁

## 使用方式

1. 使用 Pollinations AI 免费生成：
   - 访问：https://pollinations.ai/
   - 输入上述提示词
   - 下载生成的图片
   - 重命名后放入 `assets/decorations/` 目录

2. 或者使用其他 AI 绘画工具：
   - 通义万相（有免费额度）
   - 文心一格
   - Midjourney（付费）
   - Stable Diffusion（本地部署）

## 临时占位图

在正式图标生成前，可以使用：
- Emoji 表情符号
- 简单的色块
- 免费图标网站的素材（如 iconfont.cn）
