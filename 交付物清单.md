## 🐛 关键 Bug 摘要

### BUG-002（高优先级）- 必须修复
- **问题：** 多个页面使用 Vant Weapp 组件但未配置
- **影响页面：** 今日亮点、睡前故事、个人中心
- **影响：** 对话框无法显示，页面可能报错
- **建议：** 配置 Vant 组件库或改用 wx.showModal

### BUG-001（中优先级）- 建议修复
- **问题：** 首页引用不存在的页面（心情日记、成就徽章）
- **影响：** 点击后跳转失败
- **建议：** 开发页面或移除入口

### BUG-007（中优先级）- 建议修复
- **问题：** 睡前故事音频播放功能未实现
- **影响：** 故事无法播放
- **建议：** 使用 wx.createInnerAudioContext() 实现

### BUG-003（中优先级）- 建议修复
- **问题：** 聊天消息滚动功能可能异常
- **影响：** 新消息可能无法自动滚动到底部
- **建议：** 为消息项添加 id 属性
