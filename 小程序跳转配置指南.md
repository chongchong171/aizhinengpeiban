# 📱 小程序跳转配置指南

## 小红书店铺跳转配置

### 🎯 跳转目标

| 项目 | 配置 |
|------|------|
| **店铺名称** | 爻光晶舍 |
| **客服号** | `yg-crystal` |
| **跳转类型** | 微信小程序跳转 |
| **用途** | 优惠券兑换后跳转购买水晶商品 |

---

## 🔧 配置步骤

### 步骤 1：获取小红书店铺小程序 AppID

1. **打开小红书 APP**
2. **进入爻光晶舍店铺**
3. **点击店铺首页右上角「...」**
4. **查看小程序信息**
   - 记录小程序 AppID
   - 记录小程序路径（Path）

---

### 步骤 2：在微信公众平台配置跳转

1. **登录微信公众平台**
   - 网址：https://mp.weixin.qq.com
   - 扫码登录

2. **进入小程序管理后台**
   - 选择「瓜瓜陪伴」小程序

3. **配置跳转设置**
   - 菜单：设置 → 第三方设置 → 小程序跳转
   - 点击「添加跳转」
   - 填写小红书店铺小程序 AppID
   - 填写跳转路径（如：`pages/shop/index`）

4. **提交审核**
   - 跳转配置需要微信审核
   - 审核时间：1-3 个工作日

---

### 步骤 3：前端代码实现

#### 方法一：wx.navigateToMiniProgram（推荐）

```javascript
// pages/pointsShop/pointsShop.js 或 pages/myCoupons/myCoupons.js

// 跳转到小红书店铺
jumpToXiaohongshu() {
  wx.navigateToMiniProgram({
    appId: '小红书店铺 AppID', // 替换为实际 AppID
    path: 'pages/shop/index?from=guagua', // 替换为实际路径
    extraData: {
      couponId: this.data.couponId,
      userId: this.data.userId
    },
    envVersion: 'release', // trial/develop/release
    success(res) {
      console.log('跳转成功', res);
      wx.showToast({
        title: '正在打开店铺...',
        icon: 'none'
      });
    },
    fail(err) {
      console.error('跳转失败', err);
      wx.showModal({
        title: '跳转失败',
        content: '无法打开小红书店铺，请联系客服：yg-crystal',
        showCancel: false
      });
    }
  });
}
```

#### 方法二：联系客服（备用方案）

```javascript
// 如果小程序跳转失败，引导用户联系客服

contactCustomerService() {
  wx.showModal({
    title: '兑换优惠券',
    content: '请添加客服微信：yg-crystal\n发送优惠券编号获取兑换码',
    confirmText: '复制客服号',
    confirmColor: '#FF69B4',
    success(res) {
      if (res.confirm) {
        wx.setClipboardData({
          data: 'yg-crystal',
          success() {
            wx.showToast({
              title: '已复制客服号',
              icon: 'success'
            });
          }
        });
      }
    }
  });
}
```

### 步骤 4：在积分商城页面添加跳转按钮

#### 修改 `pages/pointsShop/pointsShop.wxml`

在优惠券兑换按钮旁边添加跳转按钮：

```xml
<!-- 兑换成功后显示跳转按钮 -->
<view wx:if="{{item.hasRedeemed}}" class="jump-btn" bindtap="jumpToXiaohongshu">
  <text class="btn-icon">🛍️</text>
  <text class="btn-text">去使用</text>
</view>
```

#### 修改 `pages/myCoupons/myCoupons.wxml`

在我的优惠券页面添加跳转按钮：

```xml
<!-- 优惠券卡片 -->
<view class="coupon-card" wx:for="{{coupons}}" wx:key="id">
  <view class="coupon-info">
    <text class="coupon-amount">¥{{item.amount}}</text>
    <text class="coupon-desc">{{item.description}}</text>
  </view>
  <view class="coupon-actions">
    <button class="action-btn" bindtap="jumpToXiaohongshu" data-coupon-id="{{item.id}}">
      去使用
    </button>
    <button class="action-btn secondary" bindtap="contactCustomerService">
      联系客服
    </button>
  </view>
</view>
```

## 📋 完整跳转流程

```
用户兑换优惠券
    ↓
兑换成功，显示「去使用」按钮
    ↓
点击「去使用」
    ↓
跳转到小红书爻光晶舍店铺
    ↓
用户挑选商品并下单
    ↓
下单时输入优惠券兑换码
    ↓
享受折扣
```

## ⚠️ 注意事项

### 1. 跳转资质

- 小程序跳转需要双方小程序都完成认证
- 企业主体小程序可以跳转其他小程序
- 个人主体小程序跳转受限

### 2. 审核要求

- 跳转目标小程序必须合法合规
- 不能跳转到违规内容
- 需要在小程序后台配置跳转白名单

### 3. 用户体验

- 跳转前告知用户将离开当前小程序
- 提供备用方案（如联系客服）
- 记录跳转失败日志，便于排查问题

## 🎯 备用方案

如果小程序跳转无法实现，提供以下备用方案：

### 方案 A：客服消息

```javascript
// 引导用户添加客服微信
contactService() {
  wx.showModal({
    title: '兑换说明',
    content: '1. 添加客服微信：yg-crystal\n2. 发送优惠券编号\n3. 获取兑换码\n4. 在小红书店铺使用',
    confirmText: '复制客服号',
    success(res) {
      if (res.confirm) {
        wx.setClipboardData({
          data: 'yg-crystal',
          success() {
            wx.showToast({
              title: '已复制，去微信添加',
              duration: 2000
            });
          }
        });
      }
    }
  });
}
```
