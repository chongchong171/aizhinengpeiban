# 底部导航图标说明

## 图标文件列表

需要在 `images/tab/` 目录下创建以下 8 个图标文件（4 个普通状态 + 4 个选中状态）：

### 1. 首页图标
- **home.png** - 首页图标（灰色 #999999）
- **home-active.png** - 首页图标（粉色 #FF6B81）
- **尺寸**: 81x81 像素（推荐）
- **样式**: 简洁的房屋图标

### 2. 任务图标
- **task.png** - 任务图标（灰色 #999999）
- **task-active.png** - 任务图标（粉色 #FF6B81）
- **尺寸**: 81x81 像素
- **样式**: 清单或文档图标

### 3. 商城图标
- **shop.png** - 商城图标（灰色 #999999）
- **shop-active.png** - 商城图标（粉色 #FF6B81）
- **尺寸**: 81x81 像素
- **样式**: 购物袋或商店图标

### 4. 我的图标
- **profile.png** - 我的图标（灰色 #999999）
- **profile-active.png** - 我的图标（粉色 #FF6B81）
- **尺寸**: 81x81 像素
- **样式**: 用户头像或个人图标

## 临时解决方案

如果暂时没有图标文件，可以先使用微信小程序默认的图标，或者：

1. 使用 emoji 作为临时图标（不推荐）
2. 从图标库下载免费图标（推荐）
3. 使用设计工具（如 Figma、Sketch）设计简单图标

## 推荐图标资源

- **IconFont**: https://www.iconfont.cn/
- **Flaticon**: https://www.flaticon.com/
- **Icons8**: https://icons8.com/

搜索关键词：home, task, shop, profile, user

## 图标制作工具

- **在线工具**: https://convertio.co/zh/png-ico/
- **Photoshop**: 导出为 PNG-24，透明背景
- **Figma**: 免费设计工具

## 注意事项

1. 图标尺寸建议为 81x81 像素
2. 使用透明背景
3. 保持风格统一
4. 颜色要与 tabBar 配置一致：
   - 未选中：#999999
   - 选中：#FF6B81
