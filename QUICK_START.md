# 🚀 瓜瓜陪伴小程序 - 快速启动指南

## ⚠️ 重要说明

**当前版本需要以下配置才能正常运行：**

### 1. 构建 npm（必须）
```
微信开发者工具 → 工具 → 构建 npm
✓ 勾选"将 JS 编译成 ES5"
```

### 2. 云开发初始化（必须）
```
1. 点击"云开发"按钮
2. 创建云环境（免费版即可）
3. 复制环境 ID
4. 修改 app.js 中的 env: 'your-env-id'
```

### 3. 创建数据库集合
```
云开发控制台 → 数据库 → 创建以下集合：
- dormitory（宿舍信息）
- dormitoryMember（宿舍成员）
- dormitoryPost（宿舍动态）
- goodnightCheckin（晚安打卡）

权限设置：所有用户可读写
```
