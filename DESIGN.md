## 📐 使用建议

### 图标使用
```css
/* 小程序中建议尺寸 */
.icon {
  width: 81px;
  height: 81px;
  object-fit: contain;
}
```

### 背景图使用
- 原图尺寸 720x1280，适合大多数手机屏幕
- 如需全屏覆盖，可使用 `background-size: cover`
- 建议添加半透明遮罩层以提升文字可读性

### 晚安图使用
- 预留文字区域：图片中下部 1/3 处
- 建议文字颜色：白色或浅黄色
- 可添加半透明黑色遮罩（opacity: 0.3）提升文字对比度
